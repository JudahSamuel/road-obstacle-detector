# Smart Obstacle Detection System 🤖

## 📌 Problem
Visually impaired individuals face difficulty navigating safely in real environments.

## 💡 Solution
A real-time obstacle detection system using AI that provides voice feedback about obstacles.

## ⚙️ Features
- Real-time object detection  
- Voice alerts  
- Focus on relevant obstacles only  

## 🛠 Tech Stack
Python, OpenCV, YOLO, pyttsx3  

## 🚀 Status
🟢 In Progress / Prototype Ready
